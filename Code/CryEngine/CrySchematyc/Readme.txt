Schematyc allows designers to contstruct and script new objects and entities.
The system was built primarily for the Hunt project, with the long term intention of sharing it with other projects once proven and stable.

Feature page can be found here: https://crywiki/display/CryENGINE/FP+Engine+Extension+-+Schematyc
WIP documentation can be found here: https://crywiki/display/GV/Schematyc
